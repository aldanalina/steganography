# Практикалық жұмыс №12. Желілік стеганографияны анықтау

Бұл жұмыста компьютерлік желілерде стеганография әдістерін анықтау үшін Python бағдарламасы жасалды. `network_steganography.py` модулі желілік трафикті талдау функцияларын қамтиды. DNS tunneling анықтау функциясы DNS сұрауларын талдайды: ұзын субдомендерді, Base64-қа ұқсас таңбаларды және жоғары энтропияны тексеріп, күдікті сұрауларды анықтайды. Энтропия есептеу функциясы Shannon энтропиясын пайдаланып, деректердің кездейсоқтығын өлшейді - жоғары энтропия кодталған дерек белгісі болуы мүмкін. HTTP header талдауы User-Agent, Cookie, Referer сияқты өрістерді тексеріп, ұзын мәндерді, Base64-қа ұқсас форматтарды және жоғары энтропияны анықтайды. ICMP payload талдауы ping пакеттерінің мәзмұнын зерттеп, жоғары энтропиялы пакеттерді күдікті деп белгілейді. Анықтау есебін жасау функциясы барлық күдікті DNS сұраулар мен HTTP header-ларды текстовый файлға сақтайды. Бұл практика желілік стеганографияны анықтау негіздерін көрсетеді, DNS tunneling және HTTP covert channel сияқты әдістерді тануға көмектеседі.
