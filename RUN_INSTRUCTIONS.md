# 📸 Скриншот Үшін Нұсқаулар

## Қысқа Нұсқа (Ұсынылады)

Барлық практикаларды қысқаша көрсету:

```bash
./run_all_compact.sh
```

**Артықшылықтары:**
- ⚡ Жылдам (5-10 секунд)
- 📊 Тек маңызды нәтижелер
- 🎯 Скриншотқа дайын
- 🖥️ Бір экранға сыяды

## Толық Нұсқа

Барлық практикаларды толық нәтижелермен:

```bash
./run_all_practices.sh
```

**Артықшылықтары:**
- 📋 Толық нәтижелер
- 🎨 Әдемі форматталған
- ⏱️ Паузалар бар
- 📊 Толық статистика
