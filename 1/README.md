# Практика 1: ASCII Стеганография

## Мақсаты
Мәтінге деректерді енгізу әдістерін үйрену (3 әдіс)

## Қажетті Файлдар
- `cover.txt` - бастапқы мәтін
- `secret.txt` - құпия хабар

## Қолдану

```bash
python steganography_ascii.py
```

## Нәтиже
3 файл жасалады:
- `stego_cover_whitespace.txt` - бос орындар әдісі
- `stego_cover_symbol.txt` - таңбаларды алмастыру әдісі
- `stego_cover_case.txt` - регистр әдісі

## Әдістер
1. **Whitespace** - жол соңындағы пробел/табуляция
2. **Symbol Replacement** - қарапайым/ажырамайтын пробел
3. **Case** - әріп регистрі (үлкен/кіші)
