 Мақсаты
Студенттерді мәтіндік стеганография принциптерімен таныстыру
ASCII кодтаудың мүмкіндіктерін пайдаланып, жасырын хабарды енгізу және шығарып алуды үйрету
Қажетті құралдар
Кез келген мәтіндік редактор (Notepad++, VS Code)
Python (автоматтандыру үшін, міндетті емес)

Қадамдық нұсқаулық
1️. Бастапқы деректерді дайындау
cover.txt атты мәтіндік файл жасаңыз және оған кез келген абзац жазыңыз (ағылшын немесе қазақ тілінде)
Құпия хабарды secret.txt файлына жазыңыз, мысалы:
Kezdesu_sagat_17_00

2️. 1-әдіс: Жол соңындағы бос орындарды пайдалану (Whitespace Steganography)
Ендіру:
Құпия хабардағы әр әріпті ASCII → екілік кодқа айналдырыңыз
Мысалы: M → 01001101
Мәтін жолдарының соңына қосымша таңба қойыңыз:
Пробел = 0
Табуляция = 1
Файлды жаңа атпен сақтаңыз (stego_cover.txt).
Шығару:
Файлды жол бойынша оқыңыз
Соңындағы пробел/табуляцияларды сканерлеңіз, екілік код жинаңыз
Екілік кодты ASCII символдарға түрлендіріп, хабарды қалпына келтіріңіз

3️. 2-әдіс: Таңбаларды алмастыру (Symbol Replacement)
Принцип:
Қарапайым пробел (0x20) = бит 0
Ажырамайтын пробел (0xA0) = бит 1
Ендіру:
Құпия хабарды екілік кодқа аударыңыз
Мәтіндегі пробелдерді қажетті жерінде ажырамайтын пробелге ауыстырыңыз (бит = 1 болса)
Файлды сақтаңыз
Шығару:
Әр пробелді тексеріңіз
Егер 0xA0 болса → 1, ал егер 0x20 болса → 0
Жиналған биттерді байтқа топтастырып, ASCII символдарын қалпына келтіріңіз

4️. 3-әдіс: Әріп регистрін пайдалану (Case Steganography)
Принцип:
Кіші әріп = 0
Үлкен әріп = 1
Ендіру:
Құпия хабарды екілік кодқа айналдырыңыз
Мәтіндегі әріптердің регистрін битке сәйкес өзгертіңіз
Бит = 1 → әріпті бас әріп қылыңыз
Бит = 0 → әріпті кіші қалдырыңыз
Шығару:
Әріптерді ретімен оқыңыз
Егер әріп бас болса → 1, кіші болса → 0
Биттерді жинап, хабарды шығарыңыз

5️. Нәтижені тексеру
Бастапқы және модификацияланған мәтінді салыстырыңыз – визуалды айырмашылық аз
Құпия хабарды шығарып, бастапқы мәтінге сәйкестігін тексеріңіз
