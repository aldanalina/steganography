# Практикалық жұмыс №6. Құжаттарға (PDF, DOCX, ZIP) жасырын деректерді енгізу

Бұл жұмыста әртүрлі файл форматтарына (PDF, DOCX, ZIP) жасырын деректерді енгізу және шығару әдістері Python тілінде зерттелді. `document_steganography.py` скрипті үш негізгі бағытты қамтиды. PDF файлдар үшін metadata өрістеріне (Author, Title, Subject) жасырын хабарларды Base64 форматта енгізу және PyPDF2 кітапханасы арқылы шығару мүмкіндігі жасалды. DOCX файлдар үшін custom XML properties қолданылады - python-docx кітапханасы арқылы құжат қасиеттеріне жасырын деректер қосылады және оқылады. ZIP архивтерге жасырын файлдарды қосу тәсілі қарастырылды - zipfile модулі арқылы архивке қосымша файл енгізіледі және кейін шығарылады. Барлық үш әдіс те файлдардың негізгі функционалдығын бұзбайды және metadata немесе қосымша құрылымдарды пайдаланады. Скрипт `hide_in_pdf()`, `reveal_from_pdf()`, `hide_in_docx()`, `reveal_from_docx()`, `hide_in_zip()` және `reveal_from_zip()` функцияларын қамтиды. Бұл практика құжат форматтарындағы жасырын каналдарды пайдалану принциптерін көрсетеді және әртүрлі файл типтерінде стеганографияның қолдану аясын кеңейтеді.

Нәтиже:

Құжат стеганографиясы сәтті жүзеге асырылды:
- PDF metadata арқылы жасырын хабар енгізілді және шығарылды
- DOCX custom properties қолданылды
- ZIP архивке жасырын файл қосылды
- Барлық файлдар қалыпты ашылады және жұмыс істейді
- Жасырын деректер metadata арқылы анық емес
- Әр әдіс үшін тест файлдары жасалды және тексерілді
