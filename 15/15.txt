Практикалық жоба. Ақпаратты жасырын пайдалану және оны қорғау стратегиясы жасау

1. Мақсаты
Ақпаратты жасыру және қорғаудың техникалық (стеганографиялық, криптографиялық) және ұйымдық (саясаттық) тәсілдерін қолдану.
Қауіп-қатерлерді бағалау және оларды азайтуға бағытталған стратегия әзірлеу.
Нақты мысал арқылы ақпараттың жасырын қолданылу циклін (жасау → сақтау → тасымалдау → қорғау → анықтау) модельдеу.

2. Теориялық негіздеме
Ақпаратты жасырын пайдалану (Covert Data Use) – бұл ақпаратты сыртқы бақылаушылардан жасыра отырып, оны сақтау немесе беру процесі.
Бұл тәсілдің мақсаты – деректің бар екенін жасыру және мазмұнын қорғау.
Қолданылатын технологиялар:
Криптография – ақпаратты шифрлау (AES, RSA, ChaCha20).
Стеганография – ақпаратты суретке, бейнеге, аудиоға немесе желілік пакетке енгізу.
Желілік қауіпсіздік – VPN, DNS tunneling блоктау, egress фильтрация.
Қауіпсіздік саясаты (Information Security Policy) – ұйым деңгейінде жасырын деректермен жұмыс істеу ережелері.

3. Жоба құрылымы
Этап 1. Ақпаратты жасыру (шифрлау + стеганография)
Хабарды AES алгоритмімен шифрлау.
Шифрланған хабарды суретке (PNG) LSB әдісімен енгізу.
Жасырылған файлды тасымалдау (мысалы, электрондық пошта немесе бұлт арқылы).
Мысал (Python):
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
from stegano import lsb
import base64

# 1. AES шифрлау
key = get_random_bytes(16)
cipher = AES.new(key, AES.MODE_EAX)
message = b"Жасырын мәлімет: ProjectKey=AI2030"
nonce = cipher.nonce
ciphertext, tag = cipher.encrypt_and_digest(message)
data = base64.b64encode(nonce + ciphertext).decode()

# 2. Стеганография
secret_img = lsb.hide("cover.png", data)
secret_img.save("hidden.png")
print("Жасырылған сурет сақталды: hidden.png")

Этап 2. Ақпаратты шығару және қалпына келтіру
from stegano import lsb
from Crypto.Cipher import AES
import base64

# 1. Шығару
extracted = lsb.reveal("hidden.png")
data = base64.b64decode(extracted)
nonce, ciphertext = data[:16], data[16:]

# 2. Дешифрлеу
cipher = AES.new(key, AES.MODE_EAX, nonce=nonce)
plaintext = cipher.decrypt(ciphertext)
print("Қалпына келтірілген хабар:", plaintext.decode())

Этап 4. Қорғау стратегиясы
Техникалық деңгей:
Шифрлау + Стеганография комбинациясын пайдалану.
Файлдарға қолжетімділікті шектеу (ACL, RBAC).
IDS/IPS жүйелерінде стегоанализ ережелерін енгізу.
VPN және TLS қолданысы.
Ұйымдық деңгей:
Ақпараттық қауіпсіздік саясатын әзірлеу.
Деректерді жіберу/қабылдау логтарын сақтау.
Құпия деректермен жұмыс істеу тәртібін бекіту.
Оқыту және аудит өткізу.
Қалпына келтіру стратегиясы:
Резервтік көшірме сақтау.
Бақылау және инцидентке әрекет ету жоспары (IRP).
Хабар жоғалған жағдайда қалпына келтіру процедурасы.

Этап 5. Нәтиже
Стеганография мен криптографияны біріктірген жұмыс моделі іске асты.
Жасырылған ақпарат шифрланған күйде тасымалданды.
Қауіптер талданып, қорғаныс стратегиясы жасалды.

4. Қорытынды
Ақпаратты қорғаудың ең тиімді тәсілі – бірнеше деңгейлі қорғаныс (multi-layer security).
Криптография хабардың мазмұнын қорғайды, ал стеганография оның бар екенін жасырады.
Ұйымдық саясат пен техникалық құралдар үйлессе, ақпараттың құпиялығы мен тұтастығы сенімді қамтамасыз етіледі.

Студенттерге жеке практикалық тапсырмалар 
AES арқылы хабарды шифрлаңыз.
Шифрланған деректі суретке енгізіңіз.
Шифрсыз хабар енгізіп, айырмашылығын салыстырыңыз.
JPEG форматында енгізіп, хабар жоғалуын тексеріңіз.
WAV аудиоға LSB әдісімен хабар енгізіңіз.
Шифрланған хабарды WAV ішінен шығарыңыз.
Бір хабарды екі файлға бөліп енгізіңіз.
Кілтті бөлек файлда сақтап, қайта жүктеңіз.
Хабарды QR-кодқа айналдырып енгізіңіз.
AES орнына ChaCha20 қолданыңыз.
StegExpose көмегімен жасырын хабарды анықтауға тырысыңыз.
Файл көлемінің айырмасын өлшеңіз (cover.png vs hidden.png).
Кілтті өзгертсе хабар ашылмайтынын дәлелдеңіз.
Дерек жойылған жағдайда қалпына келтіруді сынаңыз.
OpenStego бағдарламасымен балама әдіс қолданып көріңіз.
Файлды бұлтқа жүктеп, хабардың сақталуын тексеріңіз.
Steganalysis құралымен LSB таралуын зерттеңіз.
AES кілт ұзындығы мен шифрлау уақыты арасындағы байланысты анықтаңыз.
Көпқабатты қорғау жүйесін (AES + LSB + ZIP) құрастырыңыз.
Бір хабарды сурет пен аудиоға қатар енгізіңіз.
Шифрланған деректі блокчейнге сақтау идеясын сипаттаңыз.
Пайдаланушы рұқсаттары бар кілт сақтау жүйесін жасаңыз.
DLP (Data Loss Prevention) жүйесіне сәйкес ескерту ережесін жасаңыз.
Қауіпсіздік саясатының жобасын жазыңыз.
Қалпына келтіру жоспарының құрылымын жасаңыз.
Хабардың энтропиясын есептеп салыстырыңыз.
Кілтсіз ашу әрекеттерінің нәтижесін талдаңыз.
Stego файлдың хэшін есептеңіз.
Python-скриптті командалық интерфейспен толықтырыңыз.
