Практикалық жұмыс №4. Кодтау алгоритмдерінің көмегімен деректерді шығару және жасыру

Мақсаты
Кодтау (encoding) алгоритмдерінің негізгі қағидаларын меңгеру.
Деректерді әртүрлі кодтау әдістері арқылы жасыру және қайта шығарып алу тәжірибесін жасау.
ASCII, Base64, XOR және Цезарь шифры сияқты қарапайым алгоритмдерді қолдана отырып, жасырын хабарламаларды енгізіп, қайта ашуды үйрену.
Ақпаратты қорғауда кодтау мен шифрлаудың айырмашылығын түсіну.

Теориялық кіріспе
Кодтау (encoding) – ақпаратты белгілі бір ережелер бойынша басқа формаға түрлендіру әдісі. Ол деректерді қате жібермей сақтау мен тасымалдауға арналған. Ал шифрлау (encryption) құпиялылықты қамтамасыз етеді. Практикада кодтау алгоритмдері көбінесе деректерді жасыру үшін қолданылады. Мысалы:
ASCII кодтау: әрбір таңбаға 7/8 биттік сан сәйкес келеді.
Base64 кодтау: бинарлық деректерді мәтінге айналдырады, интернет хаттамаларында кең қолданылады.
XOR операциясы: биттік операция арқылы деректерді жасырудың қарапайым тәсілі.
Цезарь шифры: алфавиттегі әріптерді белгілі бір қадамға жылжытып кодтау.
Бұл әдістердің басты ерекшелігі – деректерді көзге көрінбейтіндей өзгертіп, тек белгілі ережемен қайта шығарып алу мүмкіндігі.

Қадамдық нұсқаулық
Қадам 1. Python орнату
Python интерпретаторын дайындап, қажетті кітапханаларды орнатыңыз (Base64 – стандартты кітапханада бар).
Қадам 2. ASCII кодтау және шығару
message = "Құпия хабар"
ascii_codes = [ord(ch) for ch in message]
print("ASCII кодтары:", ascii_codes)
decoded = ''.join([chr(c) for c in ascii_codes])
print("Шығарылған мәтін:", decoded)
Қадам 3. Base64 арқылы жасыру және шығару
import base64
message = "SecretMessage"
encoded = base64.b64encode(message.encode())
print("Жасырын мәтін:", encoded)
decoded = base64.b64decode(encoded).decode()
print("Шығарылған мәтін:", decoded)
Қадам 4. XOR кодтау
def xor_cipher(data, key):
    return ''.join(chr(ord(c) ^ key) for c in data)
message = "HelloWorld"
key = 42
encoded = xor_cipher(message, key)
print("Жасырылған:", encoded)
decoded = xor_cipher(encoded, key)
print("Шығарылған:", decoded)
Қадам 5. Цезарь шифры
def caesar_cipher(text, shift):
    result = ""
    for ch in text:
        if ch.isalpha():
            base = ord('A') if ch.isupper() else ord('a')
            result += chr((ord(ch) - base + shift) % 26 + base)
        else:
            result += ch
    return result
text = "Crypto"
encrypted = caesar_cipher(text, 3)
decrypted = caesar_cipher(encrypted, -3)
print("Кодталған:", encrypted)
print("Шығарылған:", decrypted)

Практикада мысалдар
Мысал 1.
Base64 кодтау арқылы құпия сөзді мәтінде сақтау.
Мысал 2.
XOR операциясымен студенттің ID-н жасыру.
Мысал 3.
Цезарь шифры көмегімен қазақша сөйлемді кодтап, кейін қайта шығару.

Қорытынды
Бұл практикалық жұмыста біз кодтау алгоритмдерін пайдаланып, деректерді жасыру және шығару әдістерін тәжірибеде жасадық. Кодтау ақпаратты қорғаудың негізгі құралы емес, бірақ деректерді тасымалдауда, сақтау мен қарапайым жасырын хабарламаларда жиі қолданылады. Оны шифрлаумен бірге қолдану қауіпсіздікті арттырады.

Жеке практикалық тапсырмалар
ASCII кодтау арқылы қысқа мәтінді кодтап, қайта шығарыңыз.
Бір сөйлемді әр әрпінің ASCII кодына айналдырыңыз.
Base64 көмегімен қазақ тіліндегі сөзді кодтаңыз.
Base64 кодталған деректі интернет арқылы жіберіп, қайта шығарыңыз.
Бір суреттің атын Base64 көмегімен жасырыңыз.
XOR операциясымен екі түрлі кілт қолданып кодтап, салыстырыңыз.
XOR-ды қолдана отырып, бір файлдың мәтінін жасырыңыз.
Цезарь шифры арқылы студенттердің тізімін кодтаңыз.
Цезарь шифры қадам санын өзгертіп, нәтижені салыстырыңыз.
Қазақ әліпбиіндегі мәтінді Цезарь әдісімен кодтаңыз.
Бір хабарламаны ASCII → Base64 → XOR түрінде қатарынан кодтаңыз.
Кодталған деректерді қайта шығару алгоритмін жазыңыз.
Бір хабарламаны кодтап, сыныптасыңызға жіберіп, оның қайта шығаруын тексеріңіз.
Base64 арқылы PDF файл атауын жасырыңыз.
XOR әдісімен құпия сөзді кодтап, тек белгілі кілтпен ашылатын болсын.
Әртүрлі кодтау әдістерімен бір хабарламаны сақтап, салыстыру жасаңыз.
Base64 көмегімен эмодзи символдарын кодтаңыз.
ASCII кодтарын пайдаланып, сандарды шифрлаңыз.
XOR көмегімен қысқа әңгіме мәтінін жасырыңыз.
Цезарь шифрын пайдаланып, топтық чаттағы хабарламаны кодтаңыз.
Мәтінді Base64 кодтап, оны QR-кодқа айналдырыңыз.
ASCII кодталған хабарламаны кесте түрінде көрсетіңіз.
XOR кодталған деректерді график түрінде бейнелеңіз (бит деңгейінде).
Бір мәтінді үш түрлі кодтау әдісімен кодтап, қайсысы тиімді екенін талдаңыз.
Цезарь шифрының кодталған мәтінін brute-force әдісімен ашып көріңіз.
ASCII арқылы қазақша мәтінді кодтап, Unicode айырмашылығын байқаңыз.
Base64 арқылы аудио файл атауын жасырып көріңіз.
XOR операциясын бірнеше кілтпен қатар қолданып көріңіз.
Цезарь әдісін сандарға қолдану тәжірибесін жасаңыз.
Кодтау нәтижелерін кестеге енгізіп, қорытынды жасаңыз.
