Практикалық жұмыс №3. Мәтіндік және суреттің аралас қолданылуында жасырын деректерді шығару

Мақсаты
Мәтін мен суреттерде жасырын ақпаратты енгізу және шығару тәсілдерін тәжірибеде көрсету.
Стеганография негіздерімен танысу.
Қарапайым құралдар мен бағдарламалар арқылы мәтіндік деректерді суреттерге енгізіп, кейін қайта шығарып алу.
Ақпараттық қауіпсіздік тұрғысынан стеганографияның артықшылықтары мен осал тұстарын түсіну.

Теориялық кіріспе
Стеганография – ақпаратты тасымалдаушыға (мәтін, сурет, дыбыс, бейне) көзге байқалмайтындай етіп енгізу әдістері. Оның басты мақсаты – құпия деректердің бар екенін жасыру. Мысалы, суреттің пиксельдеріндегі ең кіші биттерге мәтіндік хабарлама кодтауға болады. Мұндай әдіс ақпараттық қауіпсіздікте хабарламаларды жасырын жеткізуге қолданылады. Бірақ шабуылшылар арнайы стегоанализ арқылы жасырын деректерді шығара алады. Практикада біз мәтін мен суретті қатар қолдана отырып, стеганографиялық деректерді енгізу және шығару жолдарын көрсетеміз.

Қадамдық нұсқаулық
Қадам 1. Құралдарды дайындау
Python орнатыңыз.
pillow және stegano кітапханаларын орнатыңыз:
pip install pillow stegano
Қадам 2. Қарапайым сурет таңдау
Форматы: PNG (себебі JPG сығылғандықтан деректер жоғалуы мүмкін).
Мысалы: sample.png файлын алыңыз.
Қадам 3. Суретке мәтін енгізу (жасырыну)
from stegano import lsb

secret = lsb.hide("sample.png", "Бұл құпия хабарлама")
secret.save("hidden.png")
➡ hidden.png суретінде көзге көрінбейтін мәтін жасырын болады.
Қадам 4. Жасырын мәтінді шығару
from stegano import lsb
message = lsb.reveal("hidden.png")
print("Жасырын мәтін:", message)
➡ Консольде: Жасырын мәтін: Бұл құпия хабарлама.
Қадам 5. Әдісті мәтіндік құжатпен біріктіру
Мәтіндік файлды (note.txt) жасаңыз.
Файлдағы жазбаны суретке енгізіп, кейін қайта шығарыңыз.
Қадам 6. Стегоанализ тәжірибесі
Бастапқы суретті (sample.png) және жасырын суретті (hidden.png) салыстырып көріңіз.
Айырмашылықты тек піксель деңгейінде ғана табуға болады.

Практикада мысалдар
Мысал 1.
Жасырын хабарлама ретінде студенттің аты-жөні мен ID енгізу.
Мысал 2.
Word құжатын .docx форматта суретке стеганография арқылы енгізу.
Мысал 3.
QR-код суретінің ішіне мәтін енгізіп, кейін QR-кодты оқу.

Қорытынды
Бұл практикалық жұмыста біз суреттерді тасымалдаушы ретінде қолданып, мәтіндік деректерді жасырын енгізіп және қайта шығардық. Стеганография ақпаратты құпия сақтау үшін пайдалы болғанымен, қауіпсіздік тұрғысынан қосымша шифрлау әдістерімен бірге қолданылуы керек.

Жеке практикалық тапсырмалар 
Мәтіндік хабарламаны PNG суретіне жасырып, қайта шығарып алыңыз.
Әртүрлі форматтағы суреттерді (PNG, BMP, JPG) салыстырып, қайсысы деректерді жақсы сақтайтынын анықтаңыз.
Бірнеше сөзден тұратын сөйлемді суретке енгізіп көріңіз.
Қазақ тіліндегі мәтінді енгізіп, қайта шығарыңыз.
Суретке 1000 таңбадан тұратын мәтінді енгізіп көріңіз.
Мәтінді енгізу алдында оны шифрлап, кейін шешіп көріңіз.
Бір суретке бірнеше құпия хабарлама енгізу мүмкіндігін сынап көріңіз.
Бірнеше суретке бөліп хабарлама жасырыңыз.
Word құжатынан алынған мәтінді енгізіп көріңіз.
Суретке енгізілген хабарламаның көлемін өлшеңіз.
Жасырын мәтінді басқа компьютерде шығарып көріңіз.
Бір студенттің аты-жөнін енгізіп, екінші студенттің оны шығарып алуын ұйымдастырыңыз.
Суретке енгізілген хабарламаны өзгертудің жолын тәжірибеде байқап көріңіз.
ASCII және Unicode таңбаларын енгізу арқылы айырмашылықты тексеріңіз.
Мәтінді енгізген суретті әлеуметтік желіге жүктеп, қайта жүктеп алыңыз (сапаға әсерін байқаңыз).
Жасырын мәтінді тек белгілі бір кілт арқылы шығарып алу әдісін қолданыңыз.
Стегоанализ әдістерін қолданып, жасырын деректердің бар-жоғын тексеріңіз.
Бір суретке бейнефайлдың кішкентай бөлігін енгізіп көріңіз.
Суретке енгізілген мәтіннің көлемін арттырғанда сапаның қалай өзгеретінін зерттеңіз.
Мәтіндік хабарламаны BMP форматында сақтап көріңіз.
Бірнеше студенттен суреттерге әртүрлі мәтін енгізуді сұрап, нәтижелерін салыстырыңыз.
Құпия хабарламаны суретке енгізіп, кейін қағазға шығарып, қайта сканерлеп оқып көріңіз.
Бір суретке қазақша, ағылшынша және орысша мәтін енгізіп көріңіз.
Суретке енгізілген хабарламаны аудио файлда қосымша сақтау тәжірибесін жасаңыз.
Жасырын мәтінді QR-код арқылы шифрлап енгізіңіз.
Бір суретке енгізілген хабарламаны интернет арқылы жіберіп, қабылдап шығарып көріңіз.
Күрделі құпиясөздерді енгізіп, қайта шығару нәтижесін тексеріңіз.
Python орнына басқа құралды (Steghide, OpenStego) пайдаланып көріңіз.
Суретке енгізілген хабарламаның орнын кездейсоқ алгоритм бойынша таңдап көріңіз.
Жоба есебін дайындап, алынған тәжірибелерді салыстырып, қорытынды жасаңыз.
