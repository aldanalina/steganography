# Практикалық жұмыс №3. Мәтіндік және суреттің аралас қолданылуында жасырын деректерді шығару

Бұл жұмыста суретке жасырын хабар енгізу және шығару үшін LSB (Least Significant Bit) стеганография әдісі Python тілінде жүзеге асырылды. `lsb_steganography.py` скрипті Pillow кітапханасын пайдаланып, сурет пиксельдерінің ең төменгі битін өзгертеді. Хабар енгізу процесінде мәтін алдымен екілік кодқа түрлендіріледі, содан кейін әр бит сурет пиксельдерінің RGB каналдарының соңғы битіне жазылады. Терминатор (00000000) қосылады, бұл хабардың соңын белгілейді. Хабарды шығару кезінде пиксельдердің соңғы биттері жиналып, терминаторға дейін оқылады және мәтінге қайта түрлендіріледі. Скрипт екі негізгі функцияны қамтиды: `hide_message()` - хабарды жасыру және `reveal_message()` - хабарды шығару. LSB әдісі суретке минималды өзгерістер енгізеді (әр пиксельге ±1), сондықтан визуалды түрде айырмашылық байқалмайды. Бұл практика цифрлық стеганографияның негізгі принциптерін көрсетеді және суреттердегі артық биттерді пайдалану арқылы деректерді жасыру мүмкіндігін түсіндіреді.
