# Скриншоттарды Қосу Нұсқаулары

## Скриншоттарды Қалай Қосуға Болады

Әр практикалық жұмыс папкасына скриншоттарды қосыңыз, олар автоматты түрде HTML және PDF есептеріне қосылады.

## Қолдайтын Форматтар
- PNG (ұсынылады)
- JPG / JPEG

## Файл Атаулары

Скриншоттарды кез келген атаумен сақтауға болады, бірақ мына атауларды қолдану ұсынылады:

### Ұсынылатын атаулар:
- `screenshot.png` - негізгі скриншот
- `result.png` - нәтиже скриншота
- `output.png` - шығару скриншота
- `test_result.png` - тест нәтижесі

### Мысал:
```
1/
  ├── screenshot.png  ← Скриншот терминалдағы нәтиже
  ├── result.png      ← Скриншот жасалған файлдар
  └── ...
```

## Қалай Жұмыс Істейді

1. Скриншотты папкаға қойыңыз (мысалы: `1/screenshot.png`)
2. `generate_html_report.py` скриптін орындаңыз:
   ```bash
   python3 generate_html_report.py
   ```
3. Скриншот автоматты түрде HTML есепке қосылады
4. PDF-ке түрлендіру кезінде скриншоттар да қосылады

## Ескертулер

- Скриншоттар папкадағы барлық PNG/JPG файлдарды табады
- Текст файлдары (1.txt, 2.txt) елемейді
- Максимум 3 скриншот әр практика үшін қосылады
- Скриншоттар base64 форматта HTML-ге енгізіледі (PDF үшін де жұмыс істейді)

## Мысал Қолдану

```bash
# 1. Скриншотты папкаға қойыңыз
cp ~/Desktop/screenshot.png 1/screenshot.png

# 2. HTML есепті жаңартыңыз
python3 generate_html_report.py

# 3. PDF-ке түрлендіру
./convert_to_pdf.sh
# немесе браузерде Print -> Save as PDF
```

## Скриншот Қайдан Алуға Болады

1. **Терминал нәтижесі**: Скриптті орындағаннан кейін терминалдағы нәтижені скриншоттаңыз
2. **Файлдар**: Жасалған файлдарды (stego_cover.txt, hidden.png т.с.с.) көрсетіңіз
3. **Графиктер**: matplotlib арқылы жасалған графиктерді (histogram_analysis.png т.с.с.)
4. **Бағдарлама интерфейсі**: Егер GUI болса, оның скриншотын алыңыз
