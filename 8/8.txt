ПРАКТИКАЛЫҚ ЖҰМЫС 8. Деректердің бүлінген немесе бұзылған жағдайда қалпына келтіру әдістері
Мақсаты:
Windows операциялық жүйесінде деректер жоғалған, жойылған немесе бұзылған кезде оларды қалпына келтірудің негізгі әдістерін меңгеру, қалпына келтіру құралдарын тәжірибе жүзінде қолдану.
Қажетті құрал-жабдықтар:
Windows 10/11 орнатылған компьютер
USB немесе сыртқы диск
Recycle Bin, File History, System Restore, Backup & Restore
Қосымша утилиталар: Recuva, EaseUS Data Recovery, Windows Recovery Environment
Теориялық негіз
Windows жүйесінде деректерді қалпына келтіру үш деңгейде орындалады:
Деңгей
Сипаттамасы
1. Локалды қалпына келтіру
Себеттен, File History немесе Shadow Copy көмегімен файлды қалпына келтіру
2. Жүйелік қалпына келтіру
System Restore көмегімен жүйелік күйді бұрынғы нүктеге қайтару
3. Толық резервтік қалпына келтіру
System Image немесе сыртқы диск арқылы толық дискіні қалпына келтіру


ПРАКТИКАЛЫҚ ТАПСЫРМА 
1. Жүйелік қалпына келтіру нүктесін жасау
«Пуск → Панель управления → Система → Защита системы» ашыңыз.
«Создать точку восстановления» басыңыз.
Атау беріңіз (мысалы: LabPoint1).
ОК басып, нүкте құрылуын күтіңіз.

2. Тесттік файлды жою
«C:\RecoveryTest» деген қалта ашыңыз.
Ішіне DataLab.txt файлын құрып, “Test for Recovery” деп жазыңыз.
Файлды жойыңыз (Shift + Delete).
Себетті тазалаңыз.
3. File History арқылы файлды қалпына келтіру
«Пуск → Параметры → Обновление и безопасность → Резервное копирование» ашыңыз.
«Добавить диск» арқылы флеш-дискті таңдаңыз.
File History функциясын қосыңыз.
«Восстановить личные файлы» ашып, жойылған файлды табыңыз.
«Восстановить» батырмасын басыңыз.
4. Recuva утилитасын пайдалану
Recuva орнатыңыз және іске қосыңыз.
“Файлдар” қалпына келтіру режимін таңдаңыз.
Сканерлеу орнын көрсетіңіз (мысалы, C:\RecoveryTest).
“Start” басып, нәтижелерді күтіңіз.
Табылған файлды басқа дискіге қалпына келтіріңіз.
 5. System Restore пайдалану
«Пуск → Восстановление → Запуск восстановления системы» таңдаңыз.
Бұрын жасалған “LabPoint1” нүктесін таңдаңыз.
«Далее → Готово» батырмасын басыңыз.
Қайта жүктелгеннен кейін жүйе параметрлері қалпына келеді.
6. System Image Backup жасау
«Панель управления → Резервное копирование и восстановление (Windows 7)» ашыңыз.
Сол жақта «Создание образа системы» таңдаңыз.
Сыртқы дискіні көрсетіп, «Далее» → «Начать резервное копирование».
Процесс аяқталған соң, System Repair Disk жасаңыз.

Бақылау сұрақтары
System Restore және File History айырмашылығы неде?
System Image Backup не үшін қажет?
Үшінші тарап утилиталарының (Recuva, EaseUS) артықшылықтары мен шектеулері қандай?
Windows Recovery Environment қандай жағдайларда қолданылады?
Қалпына келтіру кезінде қандай қауіпсіздік ережелерін сақтау қажет?

Студенттерге жеке практикалық тапсырмалар
1
File History арқылы 3 түрлі файл түрін (TXT, DOCX, JPG) қалпына келтіріңіз.
2
System Restore көмегімен соңғы жаңартуды болдырмаңыз.
3
Recycle Bin тазаланғаннан кейін файлды Recuva арқылы қалпына келтіріңіз.
4
System Image Backup жасап, оны USB-ге сақтаңыз.
5
System Protection баптауларын өшіріп, қайта қосыңыз.
6
Recuva бағдарламасында “Deep Scan” режимін қолданып көріңіз.
7
File History сақталу аралығын 10 минутқа қойыңыз.
8
Қалпына келтіру нүктесін автоматты түрде жасау параметрін тексеріңіз.
9
EaseUS Data Recovery арқылы өшірілген фотосуретті қалпына келтіріңіз.
10
“Windows.old” қалтасынан ескі құжатты қалпына келтіріңіз.
11
System Restore сәтсіз орындалған жағдайда қатені талдаңыз.
12
Shadow Copy көмегімен файлдың ескі нұсқасын қалпына келтіріңіз.
13
Windows Recovery Environment арқылы жүйені жөндеңіз.
14
“Reset this PC” функциясын қолданып, жеке файлдарды сақтап қалу опциясын таңдаңыз.
15
Recuva көмегімен флешкадан жойылған деректерді қалпына келтіріңіз.
16
File History каталогын сыртқы дискке ауыстырыңыз.
17
System Restore орындалу уақытын өлшеңіз және есеп жазыңыз.
18
System Image-ті басқа компьютерге көшіру әрекетін орындаңыз.
19
Жойылған PDF файлды қалпына келтіріп, бүтіндігін тексеріңіз.
20
Қалпына келтірілген файлдардың бүтіндігін (checksum) тексеріңіз.
21
System Recovery параметрлерін PowerShell арқылы қараңыз.
22
File History функциясын уақытша өшіріп, нәтижесін талдаңыз.
23
Recuva нәтижелерін салыстырып, “Excellent” және “Unrecoverable” айырмасын түсіндіріңіз.
24
Windows Backup орнату каталогын өзгертіңіз.
25
System Restore көмегімен вирус орнатудан кейінгі күйге оралыңыз.
26
Қалпына келтірілген жүйенің лог файлдарын қараңыз.
27
Қалпына келтіру нүктесін жою және жаңасын жасау тәжірибесін орындаңыз.
28
File History-ден тек белгілі бір қалтаны ғана қалпына келтіріңіз.
29
System Image көмегімен толық дискіні қайта орнатыңыз (виртуалды ортада).
30
Барлық қолданылған әдістер бойынша қысқаша есеп және салыстыру кестесін құрыңыз.
