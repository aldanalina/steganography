# Практикалық жұмыс №8. Деректердің бүлінген немесе бұзылған жағдайда қалпына келтіру әдістері

Бұл жұмыста Windows операциялық жүйесінде деректерді қалпына келтіру әдістері Python тілінде симуляцияланды. `data_recovery_windows.py` скрипті Windows-тың негізгі қалпына келтіру құралдарын сипаттайды және олардың принциптерін көрсетеді. Recycle Bin қалпына келтіру функциясы жойылған файлдарды іздейді және қалпына келтіру процесін симуляциялайды. File History резервтік көшірмелер жасау және қалпына келтіру логикасын көрсетеді - файлдардың нұсқаларын уақыт белгісімен сақтау. System Restore нүктелерін жасау және қалпына келтіру функциялары жүйелік параметрлер мен файлдарды белгілі бір күйге қайтару процесін имитациялайды. Shadow Copy функциясы Windows-тың Volume Shadow Copy Service принципін көрсетеді - файл жүйесінің снапшоттарын жасау. Recuva сияқты үшінші жақты құралдар симуляциясы жойылған файлдарды іздеу алгоритмін көрсетеді. Бұл практика деректерді қалпына келтірудің әртүрлі деңгейлерін түсіндіреді: қарапайым жоюдан қалпына келтіру, нұсқаларды басқару, жүйелік снапшоттар және терең қалпына келтіру құралдары.

Нәтиже:

Windows деректерді қалпына келтіру әдістері зерттелді:
- Recycle Bin - жойылған файлдарды қалпына келтіру
- File History - файл нұсқаларын басқару
- System Restore - жүйені бұрынғы күйге қайтару
- Shadow Copy - Volume Shadow Copy Service
- Recuva - терең қалпына келтіру құралы
- Барлық әдістер симуляцияланды және сипатталды
- Әр әдістің артықшылықтары мен шектеулері көрсетілді
