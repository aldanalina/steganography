# Практика 2: BMP Hex Dump Талдау

## Мақсаты
BMP файл құрылымын талдау және пиксельдік деректерді оқу

## Қажетті Файлдар
- BMP сурет файлы (мысалы: `sample.bmp`)

## Қолдану

```bash
python bmp_analyzer.py sample.bmp
```

## Нәтиже
- BMP файл тақырыбы (сигнатура, өлшем, ені, биіктігі)
- Түс палитрасы (8-биттік BMP үшін)
- Пиксельдік деректер
- Hex дамп (алғашқы 64 байт)

## Ескерту
BMP файлын өзіңіз қосыңыз немесе интернеттен жүктеңіз.
