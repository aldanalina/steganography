# Практикалық жұмыс №2. BMP файл пішімінің құрылымын талдау

Бұл жұмыста BMP (Bitmap) файл пішімінің құрылымын талдау үшін Python бағдарламасы жасалды. `bmp_analyzer.py` скрипті BMP файлын оқып, оның он алтылық дампын талдайды. Бағдарлама файл тақырыбын (header) талдайды: алғашқы екі байтты "BM" сигнатурасы ретінде тексеріп, файл өлшемін, ені мен биіктігін, bits per pixel мәнін анықтайды. DIB Header талдауы кезінде пиксель өлшемдері мен түс тереңдігі анықталады. 8-биттік BMP файлдар үшін түс палитрасын (Color Table) талдау функциясы қамтылған, мұнда әр түстің RGB мәндері шығарылады. Сонымен қатар, пиксельдік деректердің басталу орнын (Offset) анықтап, алғашқы 10 пиксельдің мәндерін оқиды. Hex дамп функциясы файлдың алғашқы байттарын он алтылық және ASCII форматта көрсетеді. Бұл практика файл құрылымын түсінуге және бинарлық деректерді талдауға көмектеседі.

Нәтиже:

Скрипт BMP файлын талдап, келесі ақпаратты шығарады:
- Сигнатура: BM (42 4D)
- Файл өлшемі, ені, биіктігі
- Түс палитрасы (8-биттік үшін)
- Пиксельдік деректердің басталу орны
- Hex дамп (алғашқы 64 байт)
